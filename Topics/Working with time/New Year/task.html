How much time is left until the New Year? Let's create a program that will find the difference between the current date and the 1st of January of the next year!

First, you need to find the current time. Knowing the current time, you can easily find the number of the next year. Then just assemble the correct date (1st of January of the next year) and find the difference.



Memory limit: 256 MB
Time limit: 8 seconds
